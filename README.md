# 🚀 Inútil.app | Contador & Cronômetro

Um gerenciador premium de estudos minimalista e funcional, projetado para ajudar estudantes a monitorar seu progresso em tempo real. Este aplicativo combina um **Cronômetro de Estudo** com um **Contador de Questões**, oferecendo métricas de precisão e persistência de dados.

![Preview do Projeto](https://images.unsplash.com/photo-1484480974693-6ca0a78fb36b?q=80&w=2072&auto=format&fit=crop)

## ✨ Funcionalidades

- **⏱️ Cronômetro Inteligente:**
  - Inicie, pause e resete o tempo de estudo.
  - **Parciais Automáticas:** Ao pausar, uma "parcial" é salva automaticamente com o horário e a duração.
  - **Persistência:** O histórico de parciais é mantido mesmo após atualizar a página.

- **📊 Contador de Questões:**
  - Registre acertos e erros com um clique.
  - **Cálculo de Precisão:** Veja sua porcentagem de acerto em tempo real.
  - **Barra de Progresso:** Feedback visual imediato do seu desempenho.

- **💾 Persistência de Dados:**
  - Utiliza `localStorage` para garantir que seus contadores e parciais não sejam perdidos ao fechar o navegador.

- **🎨 Design Premium:**
  - Interface moderna com efeito de *glassmorphism*.
  - Fundo animado com gradientes dinâmicos.
  - Totalmente responsivo para dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização avançada com variáveis, animações e Glassmorphism.
- **JavaScript (Vanilla):** Lógica de estado, manipulação de DOM e persistência local.
- **Google Fonts:** Tipografia moderna (Outfit).

## 🚀 Como Executar o Projeto

Como este é um projeto puramente *frontend*, você pode executá-lo de duas formas:

### 1. Localmente
1. Clone o repositório:
   ```bash
   git clone https://github.com/dhelly/contador-estudos.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador.

### 2. GitHub Pages
O projeto está pronto para ser publicado no **GitHub Pages**:
1. Vá até as configurações do seu repositório no GitHub.
2. Na aba **Pages**, selecione a branch `main` e a pasta `/ (root)`.
3. Clique em **Save** e aguarde o link ser gerado.
